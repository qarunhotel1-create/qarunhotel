# نظام إدارة وثائق العملاء المتقدم

## 🎉 تم تطوير نظام جديد ومتقدم لإدارة وثائق العملاء!

### ✨ الميزات الجديدة

#### 🔧 إدارة الوثائق المتقدمة
- **رفع متعدد للملفات**: إمكانية رفع عدة ملفات في نفس الوقت
- **دعم جميع أنواع الملفات**: PDF, Word, Excel, PowerPoint, الصور، وأكثر
- **السحب والإفلات**: واجهة سهلة لرفع الملفات بالسحب والإفلات
- **معاينة فورية**: عرض الوثائق مباشرة في المتصفح
- **تحميل مباشر**: تحميل الوثائق بنقرة واحدة

#### 📱 الماسح الضوئي المتقدم
- **مسح مفرد**: مسح وثيقة واحدة بسرعة
- **مسح متعدد الصفحات**: مسح عدة صفحات في وثيقة واحدة
- **جودة عالية**: دعم دقة 300 DPI وأكثر
- **معاينة المسح**: مراجعة الصفحات قبل الحفظ
- **تحسين تلقائي**: تحسين جودة الصور المسحوبة

#### 🎨 واجهة مستخدم محسنة
- **تصميم عصري**: واجهة جميلة ومتجاوبة
- **عرض بطاقات وجدول**: إمكانية التبديل بين طرق العرض
- **بحث متقدم**: بحث سريع في أسماء العملاء والوثائق
- **فلاتر ذكية**: فلترة حسب الجنسية ونوع الوثائق
- **إحصائيات تفاعلية**: عرض إحصائيات الوثائق بشكل جميل

#### 🔒 الأمان والموثوقية
- **حذف ناعم**: الوثائق المحذوفة تبقى في النظام للاسترداد
- **تتبع التغييرات**: تسجيل جميع العمليات على الوثائق
- **نسخ احتياطية تلقائية**: حفظ تلقائي للوثائق المهمة
- **تشفير الملفات**: حماية الوثائق الحساسة

### 🚀 كيفية الوصول للنظام الجديد

#### الروابط المباشرة:
- **النظام الجديد**: `http://localhost:5000/customers-new/`
- **إضافة عميل جديد**: `http://localhost:5000/customers-new/create`
- **قائمة العملاء**: `http://localhost:5000/customers-new/`

#### من القائمة الرئيسية:
1. افتح التطبيق على `http://localhost:5000`
2. اذهب إلى قسم "العملاء"
3. ستجد رابط "النظام الجديد" في أعلى الصفحة

### 📋 دليل الاستخدام

#### إضافة عميل جديد:
1. اذهب إلى `/customers-new/create`
2. املأ البيانات الأساسية (الاسم ورقم الهوية مطلوبان)
3. أضف الوثائق بإحدى الطرق:
   - **رفع ملفات**: اسحب الملفات أو انقر للاختيار
   - **مسح ضوئي**: استخدم الماسح المدمج
   - **مسح متعدد**: امسح عدة صفحات في وثيقة واحدة

#### إدارة الوثائق:
1. اذهب إلى تفاصيل العميل
2. في قسم الوثائق يمكنك:
   - **معاينة**: عرض الوثيقة في نافذة منبثقة
   - **طباعة**: طباعة مباشرة من المتصفح
   - **تحميل**: تحميل الوثيقة على الجهاز
   - **حذف**: حذف الوثيقة (حذف ناعم)

#### البحث والفلترة:
- **البحث السريع**: ابحث بالاسم أو رقم الهوية
- **فلتر الجنسية**: اختر جنسية محددة
- **فلتر الوثائق**: اعرض العملاء الذين لديهم/ليس لديهم وثائق
- **الترتيب**: رتب حسب التاريخ أو الاسم

### 🛠️ التقنيات المستخدمة

#### الخلفية (Backend):
- **Flask**: إطار العمل الرئيسي
- **SQLAlchemy**: إدارة قاعدة البيانات
- **WTForms**: معالجة النماذج
- **Pillow**: معالجة الصور
- **python-magic**: تحديد أنواع الملفات

#### الواجهة الأمامية (Frontend):
- **Bootstrap 5**: التصميم المتجاوب
- **Font Awesome**: الأيقونات
- **JavaScript ES6**: التفاعل المتقدم
- **CSS3**: التأثيرات والانيميشن
- **HTML5**: البنية الحديثة

#### الماسح الضوئي:
- **Web API**: الوصول للكاميرا والماسح
- **Canvas API**: معالجة الصور
- **File API**: التعامل مع الملفات
- **TWAIN Support**: دعم الماسحات المتقدمة

### 📊 بنية قاعدة البيانات

#### جدول الوثائق الجديد:
```sql
CREATE TABLE customer_documents (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    customer_id INTEGER NOT NULL,
    filename VARCHAR(255) NOT NULL,
    original_name VARCHAR(255) NOT NULL,
    file_type VARCHAR(50) NOT NULL,
    file_extension VARCHAR(10) NOT NULL,
    file_size INTEGER NOT NULL,
    mime_type VARCHAR(100) NOT NULL,
    document_title VARCHAR(200),
    description TEXT,
    is_scanned BOOLEAN DEFAULT 0,
    scan_method VARCHAR(50) DEFAULT 'upload',
    pages_count INTEGER DEFAULT 1,
    scan_quality VARCHAR(20) DEFAULT 'high',
    scan_resolution INTEGER DEFAULT 300,
    status VARCHAR(20) DEFAULT 'active',
    is_verified BOOLEAN DEFAULT 0,
    upload_date DATETIME DEFAULT CURRENT_TIMESTAMP,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (customer_id) REFERENCES customers (id)
);
```

### 🔧 الإعدادات والتخصيص

#### إعدادات الرفع:
- **الحد الأقصى لحجم الملف**: 50 MB
- **الأنواع المدعومة**: جميع الأنواع
- **مجلد الحفظ**: `hotel/static/uploads/customers/`

#### إعدادات المسح:
- **الدقة الافتراضية**: 300 DPI
- **الجودة**: عالية (90%)
- **التنسيق**: JPEG للصور، PDF للمستندات

#### إعدادات العرض:
- **عدد العملاء في الصفحة**: 12 (عرض بطاقات)
- **عدد الصفوف في الجدول**: 20
- **حجم المعاينة**: 800x600 بكسل

### 🚨 استكشاف الأخطاء

#### مشاكل شائعة وحلولها:

**1. لا يمكن رفع الملفات:**
- تأكد من وجود مجلد `hotel/static/uploads/customers/`
- تحقق من صلاحيات الكتابة
- تأكد من حجم الملف أقل من 50 MB

**2. الماسح الضوئي لا يعمل:**
- تأكد من تشغيل المتصفح بصلاحيات الكاميرا
- جرب استخدام HTTPS بدلاً من HTTP
- تحقق من تعريفات الماسح الضوئي

**3. المعاينة لا تعمل:**
- تأكد من وجود الملف في المجلد الصحيح
- تحقق من صلاحيات القراءة
- جرب تحديث الصفحة

**4. البحث بطيء:**
- تأكد من وجود فهارس في قاعدة البيانات
- قلل عدد النتائج المعروضة
- استخدم فلاتر أكثر تحديداً

### 📈 الأداء والتحسين

#### نصائح لتحسين الأداء:
1. **ضغط الصور**: استخدم جودة 80% للصور الكبيرة
2. **تنظيف دوري**: احذف الوثائق المحذوفة نهائياً كل فترة
3. **نسخ احتياطية**: انسخ الملفات المهمة خارج الخادم
4. **مراقبة المساحة**: تابع مساحة التخزين المتاحة

### 🔄 التحديثات المستقبلية

#### ميزات مخططة:
- **تصنيف تلقائي**: تصنيف الوثائق حسب المحتوى
- **OCR متقدم**: استخراج النص من الصور
- **تشفير متقدم**: حماية أقوى للوثائق الحساسة
- **تكامل السحابة**: رفع تلقائي للتخزين السحابي
- **تطبيق موبايل**: تطبيق للهواتف الذكية

### 📞 الدعم والمساعدة

#### في حالة وجود مشاكل:
1. تحقق من ملف السجلات: `logs/app.log`
2. راجع هذا الدليل للحلول الشائعة
3. تأكد من تحديث النظام لآخر إصدار
4. اتصل بفريق الدعم التقني

### 📝 ملاحظات مهمة

#### التوافق مع النظام القديم:
- النظام القديم لا يزال يعمل على `/customers/`
- يمكن التبديل بين النظامين بحرية
- البيانات متوافقة بين النظامين
- لا حاجة لترحيل البيانات يدوياً

#### النسخ الاحتياطية:
- يتم إنشاء نسخة احتياطية تلقائياً قبل أي تحديث
- احفظ نسخة من مجلد `instance/` دورياً
- انسخ مجلد `hotel/static/uploads/` بانتظام
