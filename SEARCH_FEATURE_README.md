# ميزة البحث عن الحجوزات في لوحة التحكم

## نظرة عامة
تم إضافة ميزة البحث الشاملة عن الحجوزات في لوحة التحكم، والتي تتيح للمستخدمين البحث عن الحجوزات النشطة باستخدام معايير متعددة.

## الميزات المضافة

### 1. البحث السريع في لوحة التحكم
- **الموقع**: في أعلى لوحة التحكم الرئيسية
- **الوظيفة**: بحث فوري أثناء الكتابة (Live Search)
- **النتائج**: تظهر في جدول مدمج أسفل مربع البحث

### 2. صفحة البحث المتقدم
- **الموقع**: `/admin/search-bookings`
- **الوظيفة**: بحث شامل مع عرض تفصيلي للنتائج
- **المميزات**: عرض جميع تفاصيل الحجز والعميل

### 3. API للبحث السريع
- **الموقع**: `/admin/api/quick-search`
- **الوظيفة**: واجهة برمجية للبحث السريع (AJAX)
- **الاستخدام**: يستخدمها البحث السريع في لوحة التحكم

## معايير البحث

يمكن البحث باستخدام أي من المعايير التالية:

### 1. رقم الحجز
- البحث الدقيق برقم الحجز
- مثال: `16` للبحث عن الحجز رقم 16

### 2. اسم العميل
- البحث الجزئي في اسم العميل
- مثال: `أحمد` للبحث عن جميع العملاء الذين يحتوي اسمهم على "أحمد"

### 3. رقم التليفون
- البحث الجزئي في رقم التليفون
- مثال: `0101` للبحث عن جميع الأرقام التي تحتوي على "0101"

### 4. رقم الهوية
- البحث الجزئي في رقم الهوية
- مثال: `12345` للبحث عن جميع الهويات التي تحتوي على "12345"

## قيود البحث

### الحجوزات النشطة فقط
البحث يقتصر على الحجوزات ذات الحالات التالية:
- `pending` - في الانتظار
- `confirmed` - مؤكد  
- `checked_in` - تم تسجيل الدخول

### الحجوزات المستبعدة
لا يتم عرض الحجوزات التالية في نتائج البحث:
- `checked_out` - تم تسجيل الخروج
- `cancelled` - ملغي

## كيفية الاستخدام

### البحث السريع
1. اذهب إلى لوحة التحكم الرئيسية
2. استخدم مربع البحث في أعلى الصفحة
3. ابدأ بكتابة كلمة البحث (حد أدنى حرفان)
4. ستظهر النتائج تلقائياً أثناء الكتابة
5. اضغط على أيقونة العين لعرض تفاصيل الحجز

### البحث المتقدم
1. اذهب إلى لوحة التحكم الرئيسية
2. اضغط على زر "البحث المتقدم"
3. أدخل كلمة البحث في الحقل المخصص
4. اضغط على زر "بحث"
5. ستظهر النتائج في جدول مفصل مع جميع المعلومات

## المعلومات المعروضة في النتائج

### البحث السريع
- رقم الحجز
- اسم العميل
- رقم التليفون
- رقم الغرفة
- نوع الحساب (عادي/ديوز)
- حالة الحجز
- المبلغ المتبقي
- رابط عرض التفاصيل

### البحث المتقدم
- جميع المعلومات السابقة بالإضافة إلى:
- رقم الهوية
- تاريخ الوصول والمغادرة
- المبلغ الإجمالي والمدفوع
- أزرار الإجراءات (عرض، تعديل، إضافة دفعة)

## الأمان والصلاحيات

- يتطلب تسجيل الدخول (`@login_required`)
- يتطلب صلاحيات لوحة التحكم (`@admin_panel_required`)
- البحث محدود بالحجوزات النشطة فقط لحماية الخصوصية

## الأداء والتحسينات

- البحث السريع يتضمن تأخير 500ms لتجنب الطلبات المتكررة
- الحد الأقصى للنتائج: 10 نتائج في البحث السريع
- استخدام AJAX لتحسين تجربة المستخدم
- مؤشرات التحميل أثناء البحث
