# تحديث نظام التاريخ والوقت

## نظرة عامة
تم تحديث نظام التاريخ والوقت في التطبيق ليكون موحد ومضبوط على توقيت مصر مع نظام 12 ساعة.

## التغييرات المطبقة

### 1. دوال التاريخ والوقت المحدثة
- **`get_egypt_now()`**: الحصول على الوقت الحالي بتوقيت مصر مع معلومات المنطقة الزمنية
- **`get_egypt_now_naive()`**: الحصول على الوقت الحالي بتوقيت مصر بدون معلومات المنطقة الزمنية
- **`format_datetime_12h()`**: تنسيق التاريخ والوقت بنظام 12 ساعة مع صباحاً/مساءً
- **`convert_utc_to_egypt()`**: تحويل التوقيت من UTC إلى توقيت مصر

### 2. فلاتر القوالب الجديدة
- **`egypt_time`**: عرض التاريخ والوقت بنظام 12 ساعة
- **`egypt_time_24h`**: عرض التاريخ والوقت بنظام 24 ساعة (للاستخدام الداخلي)
- **`egypt_date`**: عرض التاريخ فقط
- **`egypt_time_only`**: عرض الوقت فقط بنظام 12 ساعة
- **`egypt_time_with_seconds`**: عرض التاريخ والوقت مع الثواني
- **`egypt_time_only_with_seconds`**: عرض الوقت فقط مع الثواني
- **`egypt_datetime_short`**: تنسيق مختصر للجداول

### 3. النماذج المحدثة
تم استبدال جميع استخدامات `datetime.utcnow` بـ `get_egypt_now_naive` في:
- **Customer**: نموذج العملاء
- **CustomerDocument**: نموذج وثائق العملاء
- **User**: نموذج المستخدمين
- **ActivityLog**: سجل الأنشطة
- **Note**: نموذج الملاحظات
- **BookingGuest**: نموذج مرافقي الحجز
- **RoomTransfer**: نموذج نقل الغرف

### 4. القوالب المحدثة
تم تحديث القوالب لاستخدام الفلاتر الجديدة بدلاً من `strftime` المباشر:
- لوحة تحكم المستخدم
- تفاصيل الحجز
- تفاصيل العميل
- قوائم العملاء
- التقارير

## كيفية الاستخدام

### في القوالب
```html
<!-- عرض التاريخ والوقت بنظام 12 ساعة -->
{{ booking.created_at|egypt_time }}

<!-- عرض التاريخ فقط -->
{{ booking.created_at|egypt_date }}

<!-- عرض الوقت فقط -->
{{ booking.created_at|egypt_time_only }}
```

### في الكود Python
```python
from hotel.utils.datetime_utils import get_egypt_now_naive, format_datetime_12h

# الحصول على الوقت الحالي
current_time = get_egypt_now_naive()

# تنسيق التاريخ والوقت
formatted = format_datetime_12h(current_time)
```

## أمثلة على التنسيق

### نظام 12 ساعة
- `2024-01-15 1:30 صباحاً`
- `2024-01-15 12:00 ظهراً`
- `2024-01-15 6:45 مساءً`
- `2024-01-15 11:59 مساءً`

### الوقت فقط
- `1:30 صباحاً`
- `12:00 ظهراً`
- `6:45 مساءً`
- `11:59 مساءً`

## المميزات الجديدة

### ✅ توقيت مصر الصحيح
- دعم التوقيت الصيفي تلقائياً
- تحويل من UTC إلى توقيت مصر
- مربوط بتوقيت الجهاز

### ✅ نظام 12 ساعة
- عرض الوقت بصباحاً/مساءً
- سهولة القراءة للمستخدمين
- تنسيق عربي واضح

### ✅ توحيد النظام
- جميع النماذج تستخدم نفس النظام
- فلاتر موحدة للقوالب
- سهولة الصيانة والتطوير

### ✅ مرونة في العرض
- خيارات متعددة للتنسيق
- إمكانية عرض التاريخ أو الوقت منفصلين
- دعم الثواني عند الحاجة

## ملاحظات مهمة

1. **لا يؤثر على تواريخ الحجز**: تواريخ الوصول والمغادرة تبقى كما هي
2. **متوافق مع البيانات الموجودة**: النظام يتعامل مع البيانات القديمة تلقائياً
3. **أداء محسن**: تحويل التوقيت يتم مرة واحدة فقط
4. **سهولة الاستخدام**: فلاتر بسيطة وواضحة
