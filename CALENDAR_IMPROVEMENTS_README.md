# تحسينات التقويم - عرض أسماء الأيام العربية

## الملخص
تم تحسين عرض التقويم في صفحة لوحة تحكم المستخدم لإضافة أسماء الأيام العربية بجانب التاريخ.

## التحسينات المضافة

### 1. فلاتر جديدة للتاريخ العربي
تم إضافة فلترين جديدين في `hotel/utils/template_filters.py`:

#### `arabic_day_name`
- **الوظيفة**: الحصول على اسم اليوم بالعربية من كائن التاريخ
- **المثال**: `{{ date_obj|arabic_day_name }}` → "السبت"

#### `date_with_day`
- **الوظيفة**: عرض التاريخ مع اسم اليوم بالعربية
- **المثال**: `{{ date_obj|date_with_day }}` → "السبت 31/12"

### 2. تحسين عرض التقويم
تم تحديث `hotel/templates/user/dashboard.html`:

#### رأس التقويم
- **قبل**: عرض التاريخ فقط (31/12)
- **بعد**: عرض اسم اليوم + التاريخ (السبت 31/12)
- **تمييز عطلة نهاية الأسبوع**: ألوان مختلفة للجمعة والسبت

#### تحسينات التصميم
- **ألوان متدرجة**: استخدام gradients للرؤوس
- **تمييز الحالات**: ألوان واضحة للغرف المتاحة والمحجوزة
- **تفاعل محسن**: تأثيرات hover للخلايا المتاحة
- **رموز واضحة**: ✓ للمتاح، ✗ للمحجوز

### 3. تمييز أيام الأسبوع
- **الجمعة**: لون أخضر (عطلة)
- **السبت**: لون بنفسجي (عطلة)
- **باقي الأيام**: لون وردي

## كيفية الاستخدام

### في القوالب
```html
<!-- عرض اسم اليوم فقط -->
{{ date_obj|arabic_day_name }}

<!-- عرض اسم اليوم مع التاريخ -->
{{ date_obj|date_with_day }}
```

### في التقويم
التقويم الآن يعرض:
```
| الغرفة | السبت 30/08 | الأحد 31/08 | الاثنين 01/09 |
|--------|-------------|-------------|---------------|
|  101   |      ✓      |      ✗      |       ✓       |
```
