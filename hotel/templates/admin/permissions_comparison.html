مقارنة الصلاحيات

مقارنة بين صلاحيات مسؤول النظام والمستخدم العادي

مسؤول النظام (Admin)
الصلاحيات الكاملة:
  • جميع صلاحيات الحجوزات
  • جميع صلاحيات المدفوعات
  • إدارة المستخدمين
  • إدارة الغرف
  • إدارة العملاء
  • عرض التقارير
  • حذف البيانات
  • الوصول لجميع الإعدادات
الميزات الخاصة:
  • لوحة التحكم الكاملة
  • إنشاء وحذف المستخدمين
  • تعديل إعدادات النظام
  • الوصول لجميع البيانات
معلومات الدخول:
اسم المستخدم: admin
كلمة المرور: admin
المستخدم العادي (Test User)
الصلاحيات المحددة:
  • إدارة الحجوزات
  • إدارة المدفوعات
  • إدارة العملاء
  • عرض التقارير
  • إدارة المستخدمين
  • إدارة الغرف
  • حذف البيانات
  • الإعدادات المتقدمة
الميزات المتاحة:
  • عرض جميع الحجوزات
  • إنشاء حجز جديد
  • تعديل الحجوزات
  • تسجيل دخول/خروج
  • إضافة وحذف الدفعات
  • إدارة العملاء
معلومات الدخول:
اسم المستخدم: test_user
كلمة المرور: test123
مقارنة الواجهات
مسؤول النظام يرى:
  • ✅ جميع الأزرار والإجراءات
  • ✅ لوحة التحكم الكاملة
  • ✅ إدارة المستخدمين
  • ✅ جميع التقارير والإحصائيات
  • ✅ إعدادات النظام
المستخدم العادي يرى:
  • ✅ صفحة الحجوزات الكاملة
  • ✅ جميع أزرار إدارة الحجوزات
  • ✅ أزرار إضافة وحذف الدفعات
  • ✅ تسجيل دخول وخروج العملاء
  • ❌ لا يرى إدارة المستخدمين
  • ❌ لا يرى إدارة الغرف
خطوات الاختبار
1. اختبار المستخدم التجريبي:
  1. سجل خروج من حساب admin
  2. سجل دخول بـ test_user / test123
  3. اذهب إلى "الحجوزات" من القائمة
  4. تحقق من وجود جميع الأزرار
  5. جرب إضافة دفعة لأي حجز
2. مقارنة الصلاحيات:
  1. افتح صفحة الحجوزات بحساب admin
  2. لاحظ الأزرار المتاحة
  3. سجل دخول بحساب test_user
  4. افتح نفس الصفحة
  5. قارن الأزرار المتاحة
اختبار الصلاحيات إدارة المستخدمين صفحة الحجوزات تسجيل الخروج
